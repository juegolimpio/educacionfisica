# Juego Limpio

Herramientas digitales para Educación Física, en un solo archivo HTML sin dependencias externas instaladas.

## Herramientas incluidas

**Tanteador** — marcador en vivo para clase o partido.
- Voleibol: puntos por set, diferencia mínima, sets para ganar, indicador de saque
- Básquetbol: botones +1 +2 +3, cronómetro configurable
- Tanteador libre: para cualquier deporte o juego
- Función deshacer, reinicio rápido, pantalla de campeón

**Torneo Fácil** — generador de fixtures y seguimiento de campeonatos.
- Tres formatos: todos contra todos, dos fases + final, eliminación directa
- Carga de resultados y tabla de posiciones en tiempo real
- Generación de informe copiable

## Uso

Abrir en cualquier navegador. No requiere servidor, instalación ni conexión a internet.

Compatible con celular, tablet y computadora.

Los datos se guardan automáticamente en el navegador.
